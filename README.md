# Mindspace

React + Vite recreation of the Mindspace experience.

## Development

1. `npm install`
2. `npm run dev`

## Production Build

- `npm run build` outputs static assets inside `dist/`

## GitHub Pages

- `npm run deploy` builds and publishes `dist/` to the `gh-pages` branch so the site is live at [https://muhsinh.github.io/mind.space/](https://muhsinh.github.io/mind.space/)
